2よくあるご質問

データはどこに保存されますか？

データはお使いの端末内に保存されます。iCloud が有効な同じ Apple ID の端末では、Apple の CloudKit を通じて同期できます。開発者が登録データを取得・閲覧することはなく、広告・トラッキングも行っていません。

解約や返金はできますか？

モトトレは 買い切り型です。サブスクリプションではないため解約手続きは不要です。価格は App Store の表示をご確認ください。
返金については App Store のサポートへお問い合わせください: reportaproblem.apple.com

機種変更時にデータは引き継げますか？

iCloud が有効な状態で同じ Apple ID を使うと、CloudKit を通じてデータを同期できます。同期できない場合でも、データは端末内に保存されます。

iPad / Mac でも使えますか？

iPhone と iPad に対応しています（iOS / iPadOS 17.0 以降）。Mac 版の提供予定は未定です。

3 種類の計算モード（元取り型・月リセット型・コスパ型）について、詳しくは 公式サイト をご覧ください。

機能要望はどこから送れますか？

上記の「お問い合わせ」欄からメールでご連絡ください。すべて目を通し、優先度の高いものから順に対応します。
